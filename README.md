# Toib
Задание 4
Напишите программу на Python, которая считывает файл, содержащий список паролей, по одному в строке. Он проверяет каждый пароль на предмет соответствия определенным требованиям (например, не менее 8 символов, содержит как прописные, так и строчные буквы, а также хотя бы одну цифру и один специальный символ). Пароли, соответствующие требованиям, должны быть распечатаны программой.
